### Instructions to run on local machine (Windows)

    git clone https://github.com/tanmaymodi/Edu-Fare-Backend
    git cd Edu-Fare-Backend
    npm i package.json
    
(if it shows vulnerability or error run command)

    npm audit fix

Run command 

    npm start

### Dependencies
**Backend**
1. NodeJs
2. ExpressJS
3. MongoDB
4. JsonWebToken (JWT)
5. Dompurify

**Frontend**
1. Extend JavaScript (EJS)
2. Bootstrap 4
3. JavaScript
